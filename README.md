# AI-Assisted Professor Cold-Emailer

A multi-user web app built for high school and college students to draft personalized, honest, and thoughtful research outreach emails to professors.

## Tech Architecture
- **Frontend**: Next.js 15+ (App Router), TypeScript, Tailwind CSS, Lucide Icons.
- **Backend**: Next.js API Routes, NextAuth.js (Google OAuth).
- **Database**: Supabase (Postgres) with RLS.
- **AI Service**: Generic LLM client layer (Google AI / OpenAI ready).
- **Email**: Gmail API (Google Workspace / Consumer) with OAuth2.
